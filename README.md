# ✅ ToDo App

A modern and responsive ToDo Application built with React.js that helps users manage daily tasks efficiently.

## 🚀 Live Demo

https://todo-by-ajay.vercel.app

---

## 📌 Features

### Task Management
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Track pending tasks

### Smart Filtering
- View All Tasks
- View Completed Tasks
- View Pending Tasks

### Search Functionality
- Search tasks instantly

### Statistics Dashboard
- Total Tasks Counter
- Completed Tasks Counter
- Pending Tasks Counter

### Local Storage
- Tasks persist even after page refresh
- No backend required

### Dark Mode
- Toggle between Light and Dark themes

### Responsive Design
- Works on Desktop
- Works on Tablet
- Works on Mobile Devices

## ⚡ Installation

Clone the repository

```bash
git clone https://github.com/your-username/todo-app.git
```

Navigate to project folder

```bash
cd todo-app
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm start
```
